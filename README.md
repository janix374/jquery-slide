# jquery-slide
jQuery Image Slider and SVG animations.

### tools for project
jQuery, velocity.js, webpack, sass

### Installation

1. Clone the repo
```
git clone https://github.com/janix374/jquery-slide.git
```
2. Install NPM packages
```
npm install
```
3. Start project 
```
npm run start
```
4. Build project 
```
npm run build
```
